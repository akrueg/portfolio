This page looks under the hood at touch-driven event behavior. If you touch the main green <div>, it reports details on the touch events and synthetic mouse events triggered, below which is some analysis. This code-sample is a generic utility that can be used to tinker with the color of any page element. I wrote it to help with the momentous decision of what color to make my name, above. Select / and press up & down arrow keys to re-color the <span> at the top of this page. In the console (ctr+shf+i in Chrome & FireFox), the function outputs the name / hex code / rgb value as it applies each color.
