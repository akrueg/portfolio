Sample 1: Analysis of Touch Events

This is a diagnostic tool designed to help me learn about the event-driven touchscreen behavior. If the user touches the main green <div>, it reports details on the touch events and synthetic mouse events triggered, followed by some analysis / observations about touchscreen behavior.
